## 📝 Instruções de Coleta

### Para Documentos Públicos (🔗 Download direto)

```bash
# Exemplo: Coletar Lei 14.026
curl -L "https://www.planalto.gov.br/ccivil_03/_ato2019-2022/2020/lei/l14026.htm" \
  -o data/rag-docs/san:/Lei-14026-2020.html

# Ou via script
./scripts/collect-public-documents.sh
```

### Para Documentos Restritos (📧 Requerer acesso)

```bash
# Exemplo: ABNT NBR 12216
# 1. Enviar email para: abnt@abnt.org.br
# 2. Informar: "Preciso de acesso a NBR 12211-12218 para projeto RAG"
# 3. Após aprovação, fazer download em: https://www.abnt.org.br/
# 4. Salvar em: data/rag-docs/san:/NBR-12216.pdf

# BNDES
# 1. Acessar: https://www.bndes.gov.br/
# 2. Navegar: Financiamento → Saneamento
# 3. Download de documentos
# 4. Salvar em: data/rag-docs/san:/BNDES-*.pdf
```

### Para Documentos Pagos (💰 Se Disponível)

```bash
# IEEE Standards (opcional, caro)
# 1. Verificar biblioteca institucional (universidades)
# 2. Se não houver acesso, documentar como "Não Disponível"

# ICOLD Guidelines (opcional)
# 1. Contactar: https://www.icold-cigb.org/
# 2. Verificar associações locais (CBDB)
```
